### 작동 방식

1. 웹 쇼카드를 터치하면 상품명, 가격 정보를 음파신호로 송신한다.
2. 점원은 점원용 웹을 이용해 웹 쇼카드의 내용을 등록하고 수정할 수 있다.

(+) 이후 앱에서 음파신호를 수신한 후 상품명, 가격정보를 음성으로 들려준다.
